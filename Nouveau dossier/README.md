# Mon site

Fichiers présents : `index.html`, `style.css`, `script.js`, `omega-nugget.gif`.

Pour publier sur GitHub Pages :

1. Crée un dépôt sur GitHub.
2. Pousse ce dossier sur la branche `main`.
3. Dans les paramètres du dépôt (Settings) → Pages, sélectionne la branche `main` et le dossier `/ (root)`.
4. Attends quelques minutes, puis ouvre l'URL fournie par GitHub Pages.

Si la page n'apparaît pas :
- Vérifie que `index.html` est à la racine du repo.
- Vérifie que les noms des fichiers (`style.css`, `script.js`, `omega-nugget.gif`) sont bien committés.
- Vérifie la console du navigateur pour des erreurs (CSP, 404, etc.).
